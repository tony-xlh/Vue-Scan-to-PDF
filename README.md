# Scan to PDF Vue

A Vue demo to scan documents to PDF using Dynamsoft [Mobile Web Capture](https://www.dynamsoft.com/mobile-web-capture/docs/introduction/index.html).


## How to Test

```
npm install
npm run dev
```
